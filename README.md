## ⚙️ How to run the project

- #### **Clone the repository from GitHub:**

  - git clone https://github.com/RebeckaGothlin/exjobb-EcoTrip-RebeckaGothlin

- #### **Install the dependencies:**

  - npm install

- #### **Start the developer server:**
  - npm run dev
